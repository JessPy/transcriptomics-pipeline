# Siglas do SRA e significado

Este documento resume as principais siglas usadas no SRA/ENA.

- SRR — Run accession. Identifica uma execução específica de sequenciamento.
- ERR — Run accession no European Nucleotide Archive (ENA).
- DRR — Run accession no DDBJ Sequence Read Archive.
- SRX — Experiment accession. Representa um experimento de sequenciamento.
- ERX — Experiment accession no ENA.
- DRX — Experiment accession no DDBJ.
- SRS — Sample accession. Representa uma amostra biológica ou amostral.
- ERS — Sample accession no ENA.
- DRS — Sample accession no DDBJ.
- SRP — Study accession. Identifica um estudo maior de pesquisa.
- ERP — Study accession no ENA.
- DRP — Study accession no DDBJ.
- PRJNA — BioProject accession no NCBI/SRA.
- PRJEB — BioProject accession no ENA.
- PRJDB — BioProject accession no DDBJ.
- GSM — Sample accession no GEO.
- GSE — Series accession no GEO.

## Observação

No contexto do SRA, as accessions mais usadas para download de FASTQs normalmente são:

- SRR / ERR / DRR para runs
- PRJNA / PRJEB / PRJDB para BioProjects
